This post introduces Agent SOPs (Standard Operating Procedures), a standardized markdown format for defining AI agent workflows in natural language. Read the full article on the [AWS Open Source Blog](https://aws.amazon.com/blogs/opensource/introducing-strands-agent-sops-natural-language-workflows-for-ai-agents/).

## Getting Started

To start using Agent SOPs:

```bash
pip install strands-agents-sops
```

Configure your MCP server:

```json
{
  "mcpServers" : {
    "strands-agents-sops" : {
      "command" : "strands-agents-sops",
      "args" : []
    }
  }
}
```
