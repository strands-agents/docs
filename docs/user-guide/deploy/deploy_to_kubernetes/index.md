# Deploying Strands Agents to Kubernetes

Kubernetes is a container orchestration platform that automates deployment, scaling, and management of containerized applications. It provides scalability, high availability, and efficient resource management for your Strands agents.

This guide walks you through containerizing your Strands agents and deploying them to a Kubernetes cluster, both for local development environments and deployment to a cloud service provider of your choice.
